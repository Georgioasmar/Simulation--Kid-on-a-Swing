KID ON A SWING SIMULATION PROJECT by Georgio El Asmar & Georgio Yammine
=======================================================================

Requirements to Run the Project:
-------------------------------
1. Python 3.8 or higher [https://www.python.org/downloads/]
2. Required Python libraries: run [pip install pygame numpy pygame-gui]
   - pygame (2.5.0 or later)
   - numpy (1.24.0 or later)
   - pygame-gui (0.6.9 or later)

How to Run the Simulation:
-------------------------
1. Open a terminal/command prompt
2. Navigate to the project directory: cd path/to/project/folder
3. Run main.py: python main.py

Controls During Simulation:
--------------------------
- SPACEBAR: Pause/Resume
- R: Restart simulation
- F: Toggle force vectors
- +/-: Adjust force vector scale

What to Expect:
--------------
1. First you'll see a parameter input window with sliders
2. After setting parameters, click "Start Simulation"
3. The visualization will show:
   - Real-time swing motion
   - Force vectors (toggle with F)
   - Current simulation stats
   - Constant parameters on right side
